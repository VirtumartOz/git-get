# git-get

`git get` - a better way to clone and manage git repositories.

## Features

Show repo status:
- uncommitted changes
- untracked files
- push needed (master, branch1, branch2 etc.)
- pull needed
- upstream missing
- stashes?
- state (merging, rebasing, conflict, cherry picking etc)
- ahead/behind?
- submodules?
